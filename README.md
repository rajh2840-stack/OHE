# 🔥 One Hot Encoder (OHE) Project

## 📌 Overview

This project demonstrates the implementation of **One Hot Encoding**, a popular data preprocessing technique used in Machine Learning to convert categorical data into numerical format.

One Hot Encoding transforms each category into a binary vector, making it easier for machine learning models to understand and process non-numeric data.

---

## 🚀 Features

* Convert categorical variables into numerical format
* Easy implementation using Python
* Uses popular libraries like **pandas** and **scikit-learn**
* Clean and beginner-friendly code
* Suitable for real-world ML preprocessing tasks
